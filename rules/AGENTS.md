# AGENTS.md

## 层级介绍

`rules` 目录用于维护项目开发过程中必须遵守的规则和常用操作说明。这里的文档面向开发者和自动化开发助手，用于统一代码风格、开发流程、提交要求和日常命令。

## 当前层级结构

- `AGENTS.md`：说明 `rules` 目录的用途和当前层级结构。

- `常用命令.md`：记录 Rust 开发、格式化、检查、测试、文档和代码统计相关的常用命令。

- `代码规范.md`：记录 Rust 代码层面的规范，包括 Clippy 规则、中文注释要求和类型内部排版要求。

- `开发规范.md`：记录开发环境、代码质量、设计流程、测试流程和开发收尾检查规范。

- `提交规范.md`：记录 Git message 格式、提交前检查和远端推送限制。
