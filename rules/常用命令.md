## Rust 开发常用命令

### 依赖和项目信息

```powershell
# 查看 Rust 工具链版本
rustc --version
cargo --version

# 查看当前 workspace 的包信息
cargo metadata --no-deps

# 更新 Cargo.lock 中的依赖版本
cargo update
```

### 格式化和静态检查

```powershell
# 格式化整个 workspace
cargo fmt

# 只检查格式，不修改文件
cargo fmt --check

# 检查整个 workspace 是否能通过编译
cargo check --workspace

# 运行 Clippy 静态检查
cargo clippy --workspace --all-targets

# 将 Clippy warning 当成错误
cargo clippy --workspace --all-targets -- -D warnings
```

### 构建、运行和测试

```powershell
# 构建整个 workspace
cargo build --workspace

# release 模式构建
cargo build --workspace --release

# 运行入口 crate
cargo run -p entry

# 运行整个 workspace 的测试
cargo test --workspace

# 运行指定 crate 的测试
cargo test -p entry
```

### tokei 代码统计

```powershell
# 只统计 Rust 文件
tokei . --types Rust

# 输出 JSON，便于脚本处理
tokei . --output json
```

### 清理和文档

```powershell
# 清理 target 构建产物
cargo clean

# 生成本地文档
cargo doc --workspace --no-deps

# 生成并打开本地文档
cargo doc --workspace --no-deps --open
```
